# Grace College Website

## Project Overview
This is a unified web app for those who attend Grace College!

## Technologies Used
* **HTML:** Structure for all pages
* **CSS:** Custom Styling 
* **BOOTSTRAP:** Used for responsive grid layouts and navigation 
* **JAVASCRIPT:** Used for the map api integration, and paramater hadndling to allow for hyper links to link to a specific location 
* **LEAFLET.js:** API used for interactive campus map 
* **OPENWEATHER API:** Used to fetch the temperature currently in warsaw

## Features
**INTERACTIVE CAMPUS MAP:** Has custom markers that are linked throughout the page
**ACTIVITIES FOR STUDENT:** Fun things to do, and where they are located at 
**RESPONSIVE DESIGN:** Works for all devices
**LIVE WEATHER:** Can get the temperature currently in Warsaw, IN
